Bachelor of Commerce in Computer Applications (B.Com CA) is an undergraduate program that combines commerce subjects with computer applications.

  1. Program Overview :

    • Duration: Typically 3 years, divided into six semesters.

    • Eligibility : Generally requires completion of 10+2 (high school) education with a focus on commerce, though some institutions may accept students from other streams as well.

    • Objective : The program aims to provide students with a strong foundation in commerce subjects like accounting, finance, and management, while also equipping them with essential computer application skills, including programming, database management, and software development.
  2. Core Subjects :
    1. Commerce:
      • Financial Accounting
      • Business Law
      • Corporate Accounting
      • Accounting
      • Auditing
      • Taxation
      • Financial Management
    2. Computer Applications :
      • Basics of Information Technology
      • Programming Languages (like C, C++, Java)
      • Web Development
      • Database Management Systems (DBMS)
      • E-Commerce
      • Software Development
      • Computer Networks
    3. Skills Acquired
      • Technical Skills : Programming, software development, database management, web development, etc.
      • Commerce Skills : Financial analysis, auditing, taxation, management principles, etc.
      • Analytical Skills : Problem-solving, data interpretation, logical thinking.
      • Communication and Management Skills*: Business communication, project management.
    4. Career Opportunities
      • IT Sector : Software Developer, System Analyst, Web Developer.
